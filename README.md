☕ Criollo Cafe | Premium Artisanal Coffee Website

Welcome to the official repository for Criollo Cafe, a luxury digital experience dedicated to Lucknow’s premier specialty coffee house. Nestled inside the historic Le Press building on Naval Kishore Road, Hazratganj, Criollo Cafe harmonizes old-world Parisian aesthetics with state-of-the-art artisanal brewing.

This production-ready, fully responsive web application is crafted with high-end animations, a dual-theme dark/light engine, and smart integrations, designed to feel comparable to premium brands like Starbucks Reserve, Blue Tokai, and % Arabica.

✨ Features

🎨 Modern Luxury Visual Style: Beautiful cream, gold, and dark coffee color palettes optimized for retina displays.

🌓 Dynamic Dual-Theme Engine: Seamless transitions between premium Dark Mode and elegant Light Mode with absolute color harmony.

🤖 Criollo AI Concierge Assistant: Powered by the Gemini 3-Flash API directly in the browser to answer questions about brewing methods, location, timings, and reserve bookings.

📅 Real-Time Reservation Flow: A fully functional, live validation reservation system that synchronizes instantly with our database ledger.

🌟 Interactive Spotlights & Steam Particles: Custom micro-interactions including mouse-following radial glowing highlights and rising amber-gold steam sparkles.

📖 Exquisite Curation (Price-Omitted Menu): A beautiful filterable menu highlighting specialty pour-overs, fresh French viennoiseries, and gourmet continental cuisine with zero price listings to focus on sensory craftsmanship.

📸 Authentic Lucknow Media Asset Integrations: Grounded with authentic photographs of the stunning Hazratganj storefront facade at night (image_eb0562.jpg).

🛠️ Tech Stack

Frontend Framework: React 18 (Vite-bundler)

Styling: Tailwind CSS

Database & Authentication: Firebase Firestore / Firebase Auth

Artificial Intelligence: Google Generative AI (Gemini) SDK

Icons: Custom SVG Vector Asset Set
